A aplicação tem como objetivo auxiliar os usuários na gestão de suas despesas financeiras de forma simples e eficiente. 
Por meio dela, os usuários podem adicionar novas despesas, categorizá-las, visualizar relatórios e gerenciar as categorias. 
Utiliza o AsyncStorage para armazenar as informações localmente no dispositivo. 
Além disso, usa Stack e Bottom Tab Navigator, e icones para as diferentes telas.
